# Solana AI Predictive Router

A high-performance transaction router on Solana that leverages predictive execution to minimize failed swaps.

## 🚀 Senior-Level Features

- **Predictive Execution Engine**: Real-time analysis of Solana network samples (TPS) and simulated price volatility.
- **Dynamic Slippage Optimization**: AI calculates the optimal slippage based on network congestion and market health.
- **Adaptive Priority Fees**: Automatically adjusts prioritization fees to ensure transaction inclusion during high-load periods.
- **Jupiter V6 Integration**: Uses the latest aggregation protocols for deep liquidity and optimal routing.
- **Premium Dashboard**: A futuristic, glassmorphic UI providing real-time "Predictive Insights" and network confidence scores.

## 🛠️ Architecture

- **Frontend**: React 19 (Modern Hooks) + Vanilla CSS (Glassmorphism)
- **Backend**: Node.js (High Performance)
- **Execution**: Jupiter Aggregator V6
- **AI/ML**: Predictive Engine (Solana Web3 Sample Analysis)

## 📦 Getting Started

1. **Start Backend**:
   ```bash
   cd backend
   node server.js
   ```

2. **Start Frontend**:
   ```bash
   cd frontend
   npm start
   ```

3. **Operate**:
   - Connect your Phantom Wallet.
   - Click "Swap" to trigger the Predictive Analysis.
   - View real-time AI insights (TPS, Confidence, Network Status) before the swap executes.
